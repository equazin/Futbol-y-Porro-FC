# Kick Stats

App React + Vite conectada directo a Supabase.

## Requisitos

- Node.js 20+
- npm 10+

## Variables de entorno

Crear `.env` en la raiz con:

```env
VITE_SUPABASE_URL="https://TU_PROJECT.supabase.co"
VITE_SUPABASE_PUBLISHABLE_KEY="TU_ANON_KEY"
```

## Ejecutar local

```bash
npm install
npm run dev
```

Servidor local por defecto: `http://localhost:5173`

## Build

```bash
npm run build
```
